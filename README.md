# Animal Recognition CNN

This project builds a Convolutional Neural Network (CNN)
to recognize animals from images.

## Technologies Used

- Python
- TensorFlow
- Keras
- Google Colab

## Dataset

CIFAR-10 dataset was used for training.

## Features

- CNN model training
- Image preprocessing
- Model evaluation
- Real image prediction

## Future Improvements

- Use higher resolution datasets
- Apply data augmentation
- Improve model accuracy

## How to Run

Open the notebook in Google Colab and run all cells.
